# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-03

### Added
- Comprehensive README.md with full documentation
- MIT License file
- CHANGELOG.md for version tracking
- `.nojekyll` file for GitHub Pages
- Modular navigation component (`components/nav.html`)
- Mobile-responsive navigation with dropdown menus
- WordPath app showcase page
- SonicBloom app showcase page
- Legal pages (Support, Privacy Policy, Terms of Service)
- Ko-fi integration for donations
- Pulsing tagline animation on homepage

### Changed
- Improved mobile navigation UX with fixed positioning
- Unified navigation behavior across all pages
- Consistent nav bar height across all pages

### Fixed
- Desktop hover gap issue - added invisible hover bridge
- Mobile click handlers not attaching - moved initialization to page load
- Mobile submenus appearing inside nav bar - now float as popups
- Nav bar height inconsistency - removed conflicting CSS

### Documentation
- Added table of contents to README
- Added tech stack section
- Added features list
- Added browser support table
- Added setup instructions with multiple methods
- Added deployment workflow documentation
- Added contributing guidelines

## [Unreleased]
